#  How to plot Potential vs Current Graph by Arbin Data Watcher
February 4, 2026
## 1. Introduction
### 1.1 Background
Arbin DataWatcher is primarily used as a real-time data analysis and visualization tool. It allows users to monitor test data, plot curves, and export data files efficiently. It enables users to visualize and analyze key electrochemical parameters such as voltage, current, capacity, and time.
### 1.2 Purpose of This Application Note
The purpose of this application note is to provide a step-by-step guide on how to plot a **Potential vs. Current** graph using Arbin Data Watcher. This type of plot is widely used for electrochemical analysis, especially in cyclic voltammetry and polarization studies.
## 2. Step-by-Step Guide to Plot Potential vs. Current in Arbin Data Watcher
### 2.1 Open the DataWatcher Interface.
There are multiple ways to open Data Watcher. For example, if you are using MITS 10, there are three methods as follows:
  * From the **MITS 10 home screen**, click on the “Data Plot” OR “Data Export” icon

![how to plot potential vs current graph by arbin data watcher 1](https://www.arbin.com/wp-content/uploads/2026/02/how-to-plot-potential-vs-current-graph-by-arbin-data-watcher-1.webp)
  * From your **Desktop** , double-click on the DataWatcher icon

![how to plot potential vs current graph by arbin data watcher 2](https://www.arbin.com/wp-content/uploads/2026/02/how-to-plot-potential-vs-current-graph-by-arbin-data-watcher-2.webp)
  * From the **Monitor Interface** , click on the “Data Plot” OR “Data Export” icon on the Monitor Toolbar.

![how to plot potential vs current graph by arbin data watcher 3](https://www.arbin.com/wp-content/uploads/2026/02/how-to-plot-potential-vs-current-graph-by-arbin-data-watcher-3.webp)

**View the Statistics Report**
Statistical data is extracted from the **last data point of each cycle or step** and can be quickly reviewed through the Statistics Report.
1. Open the Selected Data panel of the DataWatcher home screen.
2. Right-click on the test or tests for which you want to view a statistical report.
3. From the menu, select the “Statistical Report” option
