# Overview

The Arbin DCIM feature is a state-of-the-art technology to measure battery impedance via a DC method. This is a practical substitute for the ACIM measurement method to get EIS plot while also providing additional benefits.

# Key Features

| Feature | Description |
| --- | --- |
| Built-in universal DCIM feature | Allows easy measurement of battery impedance. |
| Mid-range frequency focus | Does not require additional AC equipment and takes less than 1s test time without disturbing regular battery tests. |
| Comparison with AC method | Traditional AC method requires additional equipment and longer measurement time. |
