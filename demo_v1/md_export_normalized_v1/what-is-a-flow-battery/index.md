# Overview

A flow battery, also known as a redox flow battery, is a liquid-based rechargeable cell that uses electrolyte solutions for energy storage and release.

# Key Features

| Feature | Description |
|---------|-------------|
| Lifespan | Up to 30 years |
| Scalability | Easily scalable with larger tanks and concentrated electrolyte solutions |
| Safety | Non-flammable electrolyte material reduces risk of thermal runaway |
