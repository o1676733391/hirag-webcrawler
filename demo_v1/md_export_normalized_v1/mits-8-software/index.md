# Overview

MITS Pro is Arbin’s comprehensive battery testing software for all applications from the lab to the production floor. It provides intuitive, complete control over testing systems to generate, analyze, and manage the data needed for battery testing.

# Key Features

| Feature | Description |
| Logical programming | Menu-driven programming for control types, termination conditions, and data logging conditions. |
| Test simulation | Simulate a test directly from a text file of time-vs-current or time-vs-power data. |
| Hardware control | Control or monitor 3rd party hardware such as temperature chambers and flow battery hardware. |
| BMS communication | Communicate with an internal battery management system (BMS) via CAN bus or SM bus protocol. |
| Parallel channels | Parallel any number of channels to allow increased current handling while compiling data. |
| Measurement inputs | Many optional measurement inputs are available, such as additional reference electrodes and temperature measurements. |
