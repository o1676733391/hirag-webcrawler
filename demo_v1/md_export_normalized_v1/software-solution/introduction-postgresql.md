# Overview

Arbin integrated PostgreSQL Server, a powerful RDBMS, is designed for managing, analyzing, and securing data. It supports a wide range of functionalities and is used across various industries and applications. PostgreSQL is an open-source relational database management system (RDBMS) known for its advanced features, reliability, and adherence to SQL standards.

# Key Features

| Feature | Description |
|---------|-------------|
| Open Source | Released under the PostgreSQL License, free to use, modify, and distribute. |
| Standards Compliance | Adheres closely to SQL standards, ensuring compatibility with SQL-based applications. |
| Extensibility | Users can define custom data types, operators, and functions, and support plugins and extensions. |
| Advanced Data Types | Supports various data types including JSON, XML, arrays, and custom types. |
| Concurrency Control | Uses Multi-Version Concurrency Control (MVCC) for handling concurrent transactions. |
| ACID Compliance | Fully supports ACID properties for reliable transactions and data integrity. |
| Scalability | Scales vertically and horizontally using tools like Citus. |
| Backup and Recovery | Provides robust options including physical backups and Point-in-Time Recovery (PITR). |
| Cross-Platform | Runs on various operating systems including Windows, Linux, macOS, and Unix-like systems. |

* We recommend customers to use PostgreSQL as the default setup on Arbin MITS and set up replication for backup, query, export, plotting, and analysis.
