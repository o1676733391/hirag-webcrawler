# Overview

This document provides instructions on how to run cyclic voltammetry on a three-electrode (3E) cell using Arbin Tester, detailing hardware connections and software configurations.

# Key Features

| Feature | Description |
|---------|-------------|
| Three-electrode system | Allows separate study of electrochemical reactions on each electrode. |
| Current control | Supports both positive and negative anodic current. |
| Auxiliary voltage measurement | Enables voltage measurement between working and counter electrodes. |
