# Overview

Voltage, Temperature, and Delta-V: Indicators of Full Charge in NiMH Cells. The Nickel-Metal Hydride (NiMH) battery is a type of rechargeable battery that uses nickel oxide hydroxide (NiOOH) as the positive electrode (cathode) and a hydrogen-absorbing alloy as the negative electrode (anode).

# Key Features

| Feature | Description |
|---------|-------------|
| Battery type | Nickel-Metal Hydride (NiMH) |
| Positive electrode | Nickel oxide hydroxide (NiOOH) |
| Negative electrode | Hydrogen-absorbing alloy |
