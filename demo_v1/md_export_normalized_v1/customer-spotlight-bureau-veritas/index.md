# Overview

Bureau Veritas provides battery testing services on behalf of retailers who need to make confident decisions when selecting a battery manufacturer for their product. They run simulations to compare batteries from different suppliers and different cell formats.

# Key Features

| Feature | Description |
|---------|-------------|
| Testing standard | NASI C18.1M for different discharge profiles |
| Battery sizes | AA, AAA, C, D, 9V, and others |
| Performance comparison | Compare supplier’s battery performance against national brands |
| Test results | Numeric data and plots for performance understanding |
| Investment guidance | Comparison data helps retailers invest in the best battery and supplier |
