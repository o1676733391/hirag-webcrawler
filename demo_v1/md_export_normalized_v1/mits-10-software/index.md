# Overview

MITS 10 represents the latest evolution of Arbin's battery testing software, offering unparalleled efficiency and functionality for applications ranging from research labs to large-scale production.

# Key Features

| Feature | Description |
| Modernized UI | Features a modernized user interface with intuitive tools like drag-and-drop editing and undo/redo. |
| Sub-Schedule integration | Supports up to 500 steps for enhanced test scheduling. |
| Enhanced compatibility | Offers seamless migration from MITS 7/8 and multi-level user permissions. |
| Lua scripting | Provides advanced testing logic capabilities. |
| Advanced data handling | Supports MS SQL, PostgreSQL, and Apache Kafka for data management. |
| Real-time monitoring | Allows monitoring of up to 9 customizable plots. |
| Intelligent testing | Enables direct input of test parameters and real-time monitoring of meta-variable values (MV_UDx). |
| Third-party interface | Facilitates faster communication via TCP/IP. |
| Built-in backup | Ensures secure storage of settings and configurations for quick recovery. |
