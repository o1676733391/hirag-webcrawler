# Overview

Arbin’s next-generation Laboratory Battery Testing (LBTS) series offers industry-leading 24-bit resolution and high-precision measurements. The all-purpose tester features true bipolar circuitry for cross-zero linearity, four current ranges per test channel, and embedded MCUs for real-time calculations. Complementing this, Arbin’s Cell-Isolation Multi-Zone Thermal Chambers (MZTC) provide 16 independently controlled temperature mini-chambers.

# Key Features

| Feature | Description |
|---------|-------------|
| Precision | 100 ppm measurement with industry-leading 24-bit resolution |
| Turn-Key Solution | Fully integrated with channel and temperature cables interconnected from channel to MZTC |
| Dynamic Temperature Setpoints | Managed independently per isolated thermal chamber |
| Reduces Complexity | Simplifies training and decreases risk of operator error |
| Flexibility | Tests 18650, 21700, 26650 up to 16 A continuously, 20 A for 1 minute |
| Maximize Floor Space | Enhanced channel density in a compact footprint |
