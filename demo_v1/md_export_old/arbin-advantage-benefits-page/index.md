Software
Arbin’s MITS Pro software is regarded as the most **powerful and flexible** in the industry. It serves as the interface platform for all our testing systems, covering all energy storage testing applications, including electrochemistry, basic life cycling, and real-world simulations.
  * **Logical, menu driven** programming for control types
  * **Test simulation** directly from a test file
  * **Control or monitor** third-party hardware
  * Communicate with BMS using **CANBus or SMBus** protocols
  * **Parallel** any number of **channels**
  * Set safety limits at the **global level** for a test, and tailor precise safety limits at the **step level** when tighter control is needed

[Learn more about software](https://www.arbin.com/software/).
Data Capabilities
Logging and analyzing results data is the ultimate purpose of battery test equipment.
  * Arbin’s standard logging rate is **2000 pts/sec** , per system
  * **Quick plotting** with **customized chart templates** and data analysis using Data Watcher
  * All data is saved in the robust **SQL** database format
  * Central Monitor & Control Solution (CMCS) uses Arbin View and CTI protocols to **manage tests remotely**
  * Central Data Solution (CDS) allows you to **centralize and back up data** to a network or cloud-based server

[Learn more about Arbin’s data capabilities](https://www.arbin.com/solutions/central-data-solution-cds/).
Auxiliaries and Accessories
Battery test equipment is a key piece of hardware for any researcher working on any energy storage device. Arbin has you covered with many **additional features and accessories** necessary for various battery test applications.
  * Our **measurement auxiliaries** add additional measurement capabilities to our state-of-the-art equipment, including secondary voltage sense, auxiliary temperature measurement, and electrochemical impedance spectroscopy
  * Our **external control auxiliaries** add the ability to control or interface with external hardware or smart circuitry inside a battery pack including analog and digital input/output, 3rd party chamber interface, and CANBus and SMBus
  * Our optional **modules and accessories** can complement main test equipment with auto-calibration, multi-temperature chamber interface, uninterruptable power supply, battery holders, battery racks, and high precision shunts
