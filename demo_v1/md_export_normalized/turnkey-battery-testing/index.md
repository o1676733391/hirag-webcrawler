# Overview

All-In-One Cell Testing with integrated test channels, temperature chambers, and cell holders.

# Key Features

| Feature | Description |
|---------|-------------|
| Precision | 100 ppm measurement with industry-leading 24-bit resolution across four current ranges per test channel |
| Turn-Key Solution | Arrives fully integrated with channel and temperature cables interconnected from channel to MZTC |
| Dynamic Temperature Setpoints | Managed independently per isolated thermal chamber |
| Reduces Complexity | Simplifies training and decreases risk of operator error |
| Flexibility | Tests 18650, 21700, 26650 up to 16 A continuously, 20 A for 1 minute |
| Maximize Floor Space | Enhanced channel density in a compact footprint |
