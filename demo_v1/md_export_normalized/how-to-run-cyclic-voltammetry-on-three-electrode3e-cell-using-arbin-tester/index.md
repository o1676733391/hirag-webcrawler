# Overview

Cyclic voltammetry is a technique used to study the electrochemical reactions on each electrode of a three-electrode (3E) cell separately. This method allows researchers to analyze the contributions of each electrode to the overall battery performance.

# Principle

In a battery with two electrodes, measuring the voltage provides the potential difference between the cathode and anode. However, this does not reveal the individual contributions of each electrode. The 3E cell includes a working electrode (W), counter electrode (C), and reference electrode (R), allowing for separate analysis of the working electrode.

# Key Parameters

| Parameter | Description |
| Working electrode | The electrode where the material under study is placed. |
| Counter electrode | The electrode that completes the circuit and allows current to flow. |
| Reference electrode | The electrode used to measure the voltage against the working electrode. |
