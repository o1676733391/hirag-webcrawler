# Detailed Explanation

## Step-by-Step Guide to Plot Potential vs. Current in Arbin Data Watcher

### Open the DataWatcher Interface

There are multiple ways to open Data Watcher. For example, if you are using MITS 10, there are three methods as follows:

1. From the **MITS 10 home screen**, click on the “Data Plot” OR “Data Export” icon.
2. From your **Desktop**, double-click on the DataWatcher icon.
3. From the **Monitor Interface**, click on the “Data Plot” OR “Data Export” icon on the Monitor Toolbar.

### Selecting Data Channels

There are three main components to the DataWatcher Interface: The Toolbar, the Data Source, and the Chart Display.

1. Select the test that you want to observe.
2. Confirm that the test information (e.g., **test name, test_id, channel**, etc.) is correct.

### Customizing the Graph

- Modify Chart Template
- Adjust axis scaling
  - X-axis: Current
  - Y-axis: Potential

## Example Result

**View the Statistics Report**

Statistical data is extracted from the **last data point of each cycle or step** and can be quickly reviewed through the Statistics Report.

1. Open the Selected Data panel of the DataWatcher home screen.
2. Right-click on the test or tests for which you want to view a statistical report.
3. From the menu, select the “Statistical Report” option.

## Data Export

1. Open the Monitor Interface.
2. Left-click the name of a single channel or left-click and drag to select multiple channels.
3. Click the “Data Export” icon in the Monitor Toolbar to open DataWatcher and export respective data.
4. In the **Export** dialog box, configure the export settings according to your needs (e.g., data range, format, output location).
