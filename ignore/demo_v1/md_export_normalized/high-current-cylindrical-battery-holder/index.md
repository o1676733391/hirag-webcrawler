# Overview

Arbin’s High Current Cylindrical Battery Holder is designed for high energy density and high capacity cells, specifically the new 2170 format cell. It provides a solution for currents ranging from 5A up to 200A and allows for quick setup and connection to Arbin’s battery test equipment or other testing assemblies.

# Key Features

| Feature | Description |
|---------|-------------|
| Current range | 5A to 200A |
| Cell compatibility | 2170 or 21700 format |
| Adjustable contact probes | Yes |
