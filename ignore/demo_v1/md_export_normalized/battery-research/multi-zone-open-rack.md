# Overview

Arbin Instruments offers a Multi-Zone Open Rack, or MZOR. This is a battery holder system that features four compartments for holding battery trays. The rack provides an open environment with no isolation.

# Key Features

| Feature | Description |
|---------|-------------|
| Compartments | Four compartments for battery trays |
| Safety | Electrical connections made through connectors at the back of the compartment to increase safety and decrease risk of operator error |
| Stackable | MZOR systems are stackable using racks offered by Arbin Instruments |
