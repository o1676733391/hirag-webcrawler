# Overview

Arbin's highest power tester, offering superior energy efficiency, is ideal for heavy-duty testing.

# Key Features

| Feature | Description |
|---------|-------------|
| Test channels | From 300V to 1000V+ |
| Power per chassis | Up to 320kW |
| Maximum power | Parallel multiple cyclers to reach 1MW |
| Channel configuration | Parallel any number of channels |
| Voltage configuration | Channels may be configured in series at time of sale to increase voltage |
| Efficiency | Regenerative circuitry with >95% efficiency |
| Drive profile | High-speed drive profile simulation |
| Communication | Integration with smart communication protocols, such as CAN-FD |
