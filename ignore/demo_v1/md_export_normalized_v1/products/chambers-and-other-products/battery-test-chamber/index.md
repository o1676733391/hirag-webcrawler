# Overview

Arbin Instruments offers a comprehensive range of auxiliary modules to enhance battery testing capabilities, including voltage, temperature, and pressure inputs, as well as interfaces for external control and integration with electrochemical impedance spectroscopy.

# Key Features

| Feature | Description |
|---------|-------------|
| Auxiliary voltage inputs | 8 inputs per module, 0.02% accuracy, up to 100G Ohm input impedance |
| Auxiliary temperature inputs | 16 inputs per module, 0.2~1°C accuracy, supports Type-T, Type-K thermocouples, PT-100, and 10k Ohm thermistors |
| Auxiliary pressure inputs | 16 channels, 0 ~ 100mV or 0 ~ 10mV pressure sensor output voltage range, ±0.1% FSR measurement accuracy |
| Electrochemical impedance spectroscopy (EIS) interface | Frequency range of 10μHz to 100kHz, integrates with Arbin cyclers |
| Multiple temperature chamber interface (MTCI) | Controls third-party temperature chambers, RS232 compliance |
| CAN Bus | Communicates with battery management systems, supports CAN 2.0 and CAN FD protocols |
| Analog output and digital input/output | 8 analog outputs and 16 digital I/O per module, supports analog 0-10V signals |
