# Overview

Batteries commonly have two electrodes: anode and cathode. A three-electrode cell includes a reference electrode (RE) that allows for greater analysis of battery performance by decoupling test results between anode and cathode.

# Key Features

| Feature | Description |
|---------|-------------|
| Reference electrode | Allows for greater analysis of battery performance. |
| Three-electrode testing | Helps identify which electrode limits cell performance. |
| Safety evaluation | Beneficial for monitoring lithium plating on anode during fast-charging. |
