# Overview

Arbin Instruments offers a full complement of auxiliary modules to address further testing requirements, expanding testing capability beyond the main charge/discharge control circuitry.

# Key Features

| Feature | Description |
|---------|-------------|
| Auxiliary voltage inputs | 8 inputs per module, 0.02% accuracy, input impedance up to 100G Ohm |
| Auxiliary temperature inputs | 16 inputs per module, 0.2~1°C accuracy, supports Type-T, Type-K thermocouples, PT-100, and 10k Ohm thermistors |
| Auxiliary pressure inputs | 16 channels, ±0.1% FSR measurement accuracy, 15V output supply to sensor |
| Electrochemical impedance spectroscopy (EIS) interface | Frequency range of 10μHz to 100kHz, integrates with Arbin cycler |
| Multiple temperature chamber interface (MTCI) | Controls third-party temperature chambers, RS232 compliance |
| CAN Bus | Communicates with battery management systems, supports CAN 2.0 and CAN FD protocols |
| Analog output and digital input/output | 8 analog outputs and 16 digital I/O per module |
