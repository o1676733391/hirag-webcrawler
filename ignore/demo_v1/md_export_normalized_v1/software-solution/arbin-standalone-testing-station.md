# Overview

The Arbin Standalone Testing Station is designed for various tests and measurements, providing real-time data transmission to the MITS software.

# Key Features

| Feature | Description |
|---------|-------------|
| System Components | Includes Desktop PCs, Arbin Testing Systems, Temperature Chambers, and Monitoring and Control. |
| Software Applications | Features Kafka Service, On Cloud Backup, API Integrations, and Database System. |
| Upgrade & Downgrade Management | Supports over-the-air (OTA) firmware installation through the Web UI. |
| Automatic Configuration & Setting | Intuitive system configuration workflow for onboarding new Arbin systems. |
| Arbin Controlling Language | Allows for control of testing flow with defined output voltage, current, and power. |
