# Overview

Battery holders for various cell types including coin, cylindrical, flat, pouch, universal, and prismatic cells.

# Key Features

| Feature | Description |
|---------|-------------|
| Coin Cell | Various holders for coin cells |
| Cylindrical Cell | Holders for cylindrical cells with different current ratings |
| Flat Cell and Pouch Cell | Holders for flat and pouch cells |
| Universal Cell | Insulation battery tray for universal cells |
| Prismatic Cell | Holders for prismatic cells |
