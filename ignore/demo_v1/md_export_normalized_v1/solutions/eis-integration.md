# Overview

Arbin's integrated EIS solution combines the multi-channel, high-precision battery test performance of Arbin with advanced EIS capability.

# Key Features

| Feature | Description |
|---------|-------------|
| Multiplexing | Each EIS unit can be multiplexed across up to 64 Arbin channels |
| Frequency range | 10 μHz to 2 MHz |
| Integrated frequency range | 10 μHz to 100 kHz when integrated with Arbin cycler |
| Compatibility | Available on Arbin LBT, LBTS, and HPT systems |
