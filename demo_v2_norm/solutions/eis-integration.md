# Overview

Arbin's integrated EIS solution combines the multi-channel, high-precision battery test performance of Arbin with advanced EIS capability. Each EIS unit can be multiplexed across up to 64 Arbin channels.

# Key Features

| Feature | Description |
|---------|-------------|
| Multiplexing | Each EIS unit can be multiplexed across up to 64 Arbin channels |
| Frequency range | 10μHz to 2MHz |
| Integrated frequency range | 10μHz to 100kHz when integrated with Arbin cycler |
| Compatibility | Available on Arbin LBT, LBTS, and HPT systems |
