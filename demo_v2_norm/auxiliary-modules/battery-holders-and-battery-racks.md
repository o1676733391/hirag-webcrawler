# Overview

Arbin offers a wide range of options and systems for connecting various types of batteries for testing. From coin cell and cylindrical cell holders and trays to flat/pouch cell holders and racks, we have the full solution for your battery test setup.

# Key Features

| Feature | Description |
| Battery holder types | Coin cell, cylindrical cell, flat/pouch cell, universal cell, prismatic cell |
| Rack material | Aluminum |
| Rack options | 5-layer, 8-layer, 16-layer, or 24-layer |
