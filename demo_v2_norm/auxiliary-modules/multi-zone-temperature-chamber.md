# Overview

Arbin's Multi-Zone Temperature Chamber (MZTC) is a battery test chamber with 4 or 8 independent temperature chambers in 1 for cells up to 60A. Insulation and protection between each chamber provides greater temperature control and a safe testing environment by isolating each cell or pair of cells, as well as making connections easy.

# Key Features

| Feature | Description |
|---------|-------------|
| Independent temperature chambers | 4 or 8 chambers for enhanced testing flexibility |
| Thermal management | State-of-the-art techniques ensure stable and uniform temperature |
| Cell isolation | Prevents weak or failed cells from affecting others during testing |
| Battery tray compatibility | Over 10 different battery trays accommodate a variety of cell types |

# Notes

* Each of the 8 zones is equipped with a pressure relief valve to provide protection in the case of gassing or thermal events.
* Built-in over-temperature protection integrates with the Arbin cycler to automatically stop tests on any cell that is out of tolerance.
