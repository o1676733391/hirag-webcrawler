# Overview

Arbin Instruments offers a Multi-Zone Open Rack, or MZOR. This is a battery holder system that features four compartments for holding battery trays. The rack provides an open environment with no isolation.

# Key Features

| Feature | Description |
| Battery holder system | Features four compartments for holding battery trays. |
| Open environment | No isolation for increased accessibility. |
| Electrical connections | Made through connectors at the back of the compartment for safety. |
| Stackable | Can be stacked using racks offered by Arbin Instruments. |
