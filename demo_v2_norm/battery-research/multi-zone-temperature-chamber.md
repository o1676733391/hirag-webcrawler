# Overview

Arbin's Multi-Zone Temperature Chamber (MZTC) is a battery test chamber with 8 independent temperature chambers in 1 for cells up to 60A. Insulation and protection between each chamber provides greater temperature control and a safe testing environment by isolating each cell or pair of cells, as well as making connections easy.

# Key Features

| Feature | Description |
|---------|-------------|
| Independent temperature chambers | Eight compact chambers ensure stable and uniform temperature for all cells. |
| Thermal management | State-of-the-art techniques prevent temperature fluctuation. |
| Cell isolation | Greater independence to access and monitor devices under test. |
| Battery tray compatibility | Over 10 different battery trays accommodate a variety of cell types. |

# Test Techniques / Control Modes

| Technique | Description | Key Parameters |
|-----------|-------------|----------------|
| Temperature Control | Apply a unique temperature setpoint to each of the 8 independent mini-chambers. | Individual cell control. |
| Manual Operation | Control the chamber using Arbin's MITS Software or use the digital display and push-button control for manual operation. | Standalone operation. |

# Safety Features

| Feature | Description |
|---------|-------------|
| Pressure relief valve | Each of the 8 zones is equipped with a valve for protection in case of gassing or thermal events. |
