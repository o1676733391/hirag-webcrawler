# Notes

*Arbin’s next-generation Laboratory Battery Testing (LBTS) series offers industry-leading 24-bit resolution and high-precision measurements. The all-purpose tester features true bipolar circuitry for cross-zero linearity, four current ranges per test channel, and embedded MCUs for real-time calculations. It provides a safe testing environment by isolating single or pairs of cells so a weak or failed cell does not interrupt others and allows greater independence in monitoring the devices under test.*
