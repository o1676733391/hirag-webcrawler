# Test Techniques / Control Modes

| Technique | Description | Key Parameters |
| --- | --- | --- |
| CC - Constant Current | Designed to deliver a steady current to the battery based on the specified value. A positive current corresponds to charging, while a negative current corresponds to discharging. |  |
| CV - Constant Voltage | Regulates the voltage output to the battery according to the set value, ensuring voltage stability during charging. |  |
| CCCV - Constant Current & Constant Voltage | Facilitates an integrated charging process, transitioning from CC to CV mode upon reaching a predefined voltage threshold. |  |
