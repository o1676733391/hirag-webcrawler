# Test Techniques / Control Modes

| Technique | Description | Key Parameters |
| --- | --- | --- |
| CC - Constant Current | Delivers a steady current to the battery based on the specified value. | Positive current for charging, negative current for discharging. |
| CV - Constant Voltage | Regulates the voltage output to the battery according to the set value. | Ensures voltage stability during charging. |
| CCCV - Constant Current & Constant Voltage | Integrated charging process transitioning from CC to CV mode. | Efficient charging with termination criteria based on time or minimum current threshold. |
