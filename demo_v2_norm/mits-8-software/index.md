# Overview

MITS Pro is Arbin’s comprehensive battery testing software for all applications from the lab to the production floor. Developed by Arbin’s software engineers, it provides intuitive, complete control over testing systems to generate, analyze, and manage the data you need. Cycle cells, perform dynamic real-world simulations, and perform electrochemical experiments within the same user interface.

# Key Features

| Feature | Description |
| Logical programming | Menu-driven programming for control types, termination conditions, and data logging conditions. |
| Test simulation | Simulate a test directly from a text file of time-vs-current or time-vs-power data such as a drive profile (DST, FUDS, HPPC, etc.). |
| Hardware control | Control or monitor 3rd party hardware such as temperature chambers, or flow battery hardware such as pumps, flow meters, heaters, valves, etc. |
| BMS communication | Communicate with an internal battery management system (BMS) via CAN bus or SM bus protocol. |
| Parallel channels | Parallel any number of channels to allow increased current handling, while automatically compiling data. |
| Measurement inputs | Many optional measurement inputs are available, such as additional reference electrodes, temperature measurements, and more. |

# Capabilities

| Capability | Description |
| Powerful test control | Create a “test object” battery profile with pre-populated safety limits or create and fully customize testing profiles for your specific application. |
| Data analysis | View, plot, and analyze your battery test data in real-time, providing immediate information and insights. |
| Data export | Export your data into CSV or Excel format through the MITS Pro interface. |

# Integrations

| Software | Purpose |
| Data Watcher | Allows data to be viewed graphically in real time while tests are running. |
| Arbin Viewer | Allows remote access to the Arbin testers. Users can search Arbin systems on their network and view their status. |

# System Requirements

| Requirement | Value |

| Database format | All data saved in the robust SQL database format. |

# Notes

* Data Watcher comes pre-installed with the Arbin system, and users also have the license to install Data Watcher on multiple PCs.
* A Console TCP Interface (CTI) is available to enterprise clients who wish to control the Arbin hardware directly using their own TCP/IP commands or DLL interface.
* Arbin’s MITS Pro uses intelligent battery behavior checks during testing to help prevent accidents.
